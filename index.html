Web Designer Nobuyuki Tawara

埼玉県在住のWebデザイナー、田原信幸です。オンラインスクールでWeb制作・プログラミングについて学び、ダンススクール経営から憧れていたWebデザイナーへと転職活動中です。

これからさらにWeb制作の技術や知識を身につけて、Webサイト運用だけでなく、経営上の様々な問題解決のアドバイスもできるデザイナー兼コンサルタントになりたいと思っています。

素晴らしいサービスを提供し、日々努力している中小企業・個人事業の経営者の方々の力になることが目標です。

Skills

Web

HTML/CSSでのコーディングができます。JavaScript、PHP、WordPressも現在勉強中です。

Google Adwords

検索連動広告（Google Adwords）の運用実績があります。運用代行、または担当の方への運用指導・アドバイスも可能です。

Consulting

２０年以上ダンススクール経営に携わってきました。ホームページデザインを提案するだけでなく、集客や経営上の問題解決のお手伝いができます。
